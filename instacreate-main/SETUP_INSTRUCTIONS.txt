TROUBLESHOOTING
================================================================================

PROBLEM: Profiles won't launch
SOLUTION: 
- Check the ChromeDriver path in manager.py
- Make sure Chrome browser is installed
- Verify Python packages are installed

PROBLEM: "Connection refused" errors
SOLUTION:
- Make sure both servers are running
- Check if ports 4000 and 5173 are free
- Restart both servers

PROBLEM: Proxy not working
SOLUTION:
- Verify proxy format is correct
- Test proxy connection separately
- Check proxy server is online
